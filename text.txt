Crear repositorio
git init

Preparar archivos
git add .

Agregar commit
git commit -m "Mi comentario"

Agregar remoto
git remote add origin urlDelServidor

Enviar al servidor 
git push -u origin nombreDelBranch

Clonar
remote clone urlDelServidor

Local
git clone path

Crear un branch 
git checkout -b nombreDelBranch 

Cambiar de branch
git checkout nombreDelBranch

Eliminar
git checkout -d nombreDelBranch

Descargar cambios
git pull

Fusionar
git merge nombreDelBranch

Comparar branch 
git diff iBranch targetBranch

En que branch estoy 
git status

Ver commits
git log

Eliminar commits previos
git reset -hard origin/branchQueNoSirveParaNada
